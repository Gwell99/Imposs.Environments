Ascii Filter 1.0

==instruction==
It's a simple image effect to make the screen looks like ascii art. It doesn't limit as ascii art. You can use custom sample texture for art style.
Just drag it to a camera and you will see the result

parameters:
Color Type: Colorful or Monochrome
Sample: sample texture, it should be adjust from right to left from bright to dark
Sample Number: how many segment does the sample texture has
Height Segment: How many pixel image should be displayed on screen, height base

==Reminder==
For better result, please set the sample texture's filter Mode as "Point". Also disable "Generate Mid Maps".
